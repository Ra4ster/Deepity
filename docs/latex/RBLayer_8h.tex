\doxysubsection{Detailed Description}
Restricted Boltzmann-\/style Predictive Coding (PC) Layer. 

Implements a single layer of a generative PC network. This layer maintains its own state beliefs and manages bidirectional errors (bottom-\/up and top-\/down). Memory is designed to be injected via a contiguous arena allocator to guarantee 64-\/byte alignment for AVX/\+\+SIMD vectorization.
