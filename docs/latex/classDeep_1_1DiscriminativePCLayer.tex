\doxysubsection{Detailed Description}
Represents a single layer in a Predictive Coding Network.
\begin{DoxyItemize}
\item To perform inference (prediction):\+
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Clamp the input data to the bottom layer\textquotesingle{}s latent state (z).
\item Enter a continuous loop across all layers, calling \doxylink{classDeep_1_1DiscriminativePCLayer_a4d50660fe8014f93db31e3478f98ee07}{Calculate\+State()} to compute the local prediction errors (e).
\item Call \doxylink{classDeep_1_1DiscriminativePCLayer_a40f4dbeedd94bdc369d45091bbbdf559}{Update\+State()} to adjust the latent states (z) based on those errors.
\item Repeat steps 2 and 3 until the states settle into an equilibrium (the energy is minimized and dz/\+dt approaches zero).
\item Read the final predictions from the latent states of the desired layers.
\item (For learning):\+ Call \doxylink{classDeep_1_1DiscriminativePCLayer_a39631620cf0d84cc97977b619fac2e9f}{Update\+Weights()} on all layers simultaneously once the states have fully settled. 
\end{DoxyEnumerate}
