\doxysubsection{Detailed Description}
\doxylink{classopenrand_1_1Philox}{Philox} generator. 

\begin{DoxyNote}{Note}
This is a modified version of \doxylink{classopenrand_1_1Philox}{Philox} generator from Random123 library. This uses 4x 32-\/bit counter, 2x 32-\/bit key along with 10 rounds. 
\end{DoxyNote}
