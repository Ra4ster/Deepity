\doxysubsection{Detailed Description}
Defines the activation functions of a predictive coding model. 

This header includes implementations of Re\+LU and TanH.

Usage:\+ \+\#include \texorpdfstring{$<$}{<}\doxylink{Activations_8h}{Activations.\+h}\texorpdfstring{$>$}{>}

Example:\+ Deep::\+tanh(array, arraysize)

\begin{DoxyNote}{Note}
Separate implementations exist for AVX512F, AVX2, SSE, and naive.
