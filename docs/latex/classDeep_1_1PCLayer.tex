\doxysubsection{Detailed Description}
Represents a single layer in a Predictive Coding Network.
\begin{DoxyItemize}
\item To perform inference (prediction):\+
\end{DoxyItemize}
\begin{DoxyEnumerate}
\item Clamp the input data to the bottom layer\textquotesingle{}s latent state (z).
\item Enter a continuous loop across all layers, calling \doxylink{classDeep_1_1PCLayer_a61b31f7447140c535123f793db549ac5}{Calculate\+State()} to compute the local prediction errors (e).
\item Call \doxylink{classDeep_1_1PCLayer_a375092aaa22c17a0c1822da96a112e3a}{Update\+State()} to adjust the latent states (z) based on those errors.
\item Repeat steps 2 and 3 until the states settle into an equilibrium (the energy is minimized and dz/\+dt approaches zero).
\item Read the final predictions from the latent states of the desired layers.
\item (For learning):\+ Call \doxylink{classDeep_1_1PCLayer_afe6a1b75f8ada9f5a1d9640573058c86}{Update\+Weights()} on all layers simultaneously once the states have fully settled. 
\end{DoxyEnumerate}
