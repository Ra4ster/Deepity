172/*
173 * MHSA USING PCNs:
174 * In a standard Transformer, Multi-Head Self-Attention (MHSA) computes Queries, Keys,
175 * and Values in a rigid, one-shot forward pass to route token context. If implemented
176 * using a PCN, the attention scores and context vectors would become dynamic latent
177 * states (z). When a sequence is input, the attention routing wouldn't be calculated
178 * instantly; instead, it would iteratively settle over artificial time. The network
179 * would adjust its attention states to minimize the prediction error between the
180 * bottom-up token data and the top-down expectations from higher layers. The Q, K,
181 * and V projection matrices would serve as the local weights (W) for this mechanism,
182 * and would be updated entirely in parallel via local Hebbian rules once the attention
183 * routing reached its minimum-energy equilibrium.
184 */
