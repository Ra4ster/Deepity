213/*
214 * MHSA USING PCNs:
215 * In a standard Transformer, Multi-Head Self-Attention (MHSA) computes Queries, Keys,
216 * and Values in a rigid, one-shot forward pass to route token context. If implemented
217 * using a PCN, the attention scores and context vectors would become dynamic latent
218 * states (z). When a sequence is input, the attention routing wouldn't be calculated
219 * instantly; instead, it would iteratively settle over artificial time. The network
220 * would adjust its attention states to minimize the prediction error between the
221 * bottom-up token data and the top-down expectations from higher layers. The Q, K,
222 * and V projection matrices would serve as the local weights (W) for this mechanism,
223 * and would be updated entirely in parallel via local Hebbian rules once the attention
224 * routing reached its minimum-energy equilibrium.
225 */
