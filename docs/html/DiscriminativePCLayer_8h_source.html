190/*
191 * MHSA USING PCNs:
192 * In a standard Transformer, Multi-Head Self-Attention (MHSA) computes Queries, Keys,
193 * and Values in a rigid, one-shot forward pass to route token context. If implemented
194 * using a PCN, the attention scores and context vectors would become dynamic latent
195 * states (z). When a sequence is input, the attention routing wouldn't be calculated
196 * instantly; instead, it would iteratively settle over artificial time. The network
197 * would adjust its attention states to minimize the prediction error between the
198 * bottom-up token data and the top-down expectations from higher layers. The Q, K,
199 * and V projection matrices would serve as the local weights (W) for this mechanism,
200 * and would be updated entirely in parallel via local Hebbian rules once the attention
201 * routing reached its minimum-energy equilibrium.
202 */
