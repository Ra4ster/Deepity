void RBLayer::UpdateBeliefs ( const float * bottomUp,
const float * topDown,
size_t currentBatchSize )
noexcept

Updates internal state beliefs via gradient descent on the prediction error.

Integrates the state dynamics using the forward error, backward error, and a Gaussian prior (decay):

\[r \leftarrow r + k_1 \left( \frac{1}{\sigma^2} (I - rU)U^T + \frac{1}{\sigma^2_{td}}(td - r) - \alpha r \right) \]

Parameters
bottomUpThe input data \( I \) from the layer below.
topDownThe target data \( td \) from the layer above (can be nullptr).
currentBatchSizeThe active batch dimension.

◆ UpdateWeights() [2/2]

void RBLayer::UpdateWeights ( size_t currentBatchSize)
noexcept

Applies batched gradients to update the synaptic weights.

Computes the weight update using the non-linear prediction error, applies L2 weight decay, and projects the rows of the weight matrix back to a unit norm:

\[\Delta U = \frac{k_2}{\sigma^2} r^T \left( (I - f(rU)) \odot f'(rU) \right) - k_2 \lambda U \]

Parameters
currentBatchSizeThe active batch dimension.
